# Profile

This is a browser profile manager made with Flutter for Windows, MacOS, and Linux.
It supports all major browsers like Firefox, Chrome, Edge etc.

## Supported Browser

|        | Firefox | Chrome | Brave | Opera | Edge
|--------|:-------:|:------:|:-----:|:-----:|:----:
|Windows | ✓      | ✓     | ✓    | ✓    | ✓
|Linux   | ✓      | ✓     | ✓    | ✓    | ✓
|MacOS   | ?       | ?      | ?     | ?     | ?
